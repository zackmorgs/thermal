## Features

- [h5bp](https://github.com/h5bp/html5-boilerplate) boilerplate in `./public`

### Instructions
- Install cmake:
    - `choco install cmake`
- Clone the repo
- Compile it:
    - `cd ./build; cmake --build .`
- Run the executable
    - `./Debug/thermal.exe "C:\path\to\your\server\root" -w`

Also available to build in Visual Studio
